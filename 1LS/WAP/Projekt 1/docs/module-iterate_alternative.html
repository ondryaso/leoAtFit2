Methods

(static) iterateProperties_iterative_fetchDescriptors()

An iterative implementation that uses Object.getOwnPropertyDescriptors. Retrieves an array of all descriptors of the object and iterates over it using Reflect.ownKeys. Collects all the property names of a single object in an array of results, uses yield* to return them and moves up the prototype chain and repeats while a prototype exists (without using recursion).

(static) iterateProperties_iterative_functional()

An iterative implementation that uses Reflect.ownKeys directly on the source object, transforms it using Array.filter and Array.map and uses yield* to return the result. Then it moves up the prototype chain and repeats while a prototype exists (without using recursion).

(static) iterateProperties_iterative_objectMethods()

An iterative implementation that uses Object.getOwnPropertyNames and Object.getOwnPropertySymbols directly on the source object, iterating over the resulting array it and yielding key-by-key. Then it moves up the prototype chain and repeats while a prototype exists (without using recursion).

(static) iterateProperties_iterative_reflectOnInput()

An iterative implementation that uses Reflect.ownKeys directly on the source object, iterating over the resulting array it and yielding key-by-key. Then it moves up the prototype chain and repeats while a prototype exists (without using recursion).

This only calls the baseline module:iterate.iterateProperties implementation. This wrapper exists for the sake of consistency and naming in the performance tests logs which uses Function.name.

(static) iterateProperties_recursive_fetchDescriptors()

A recursive implementation that uses Object.getOwnPropertyDescriptors. Retrieves an array of all descriptors of the object, iterates over it using Reflect.ownKeys, yields key-by-key and then uses yield* to call itself recursively for the prototype (if present).
